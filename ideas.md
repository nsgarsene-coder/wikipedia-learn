## Approche 1 : Archives Vivantes (Bibliothèque Encyclopédique Moderne)

**Design Movement** : New Brutalism Éditorial — fusion entre l'esthétique d'une revue académique haut de gamme et la brutalité graphique contemporaine.

**Core Principles** :
- Typographie comme architecture : les titres structurent l'espace, pas les bordures
- Contraste radical entre zones de lecture (calme, aéré) et zones d'action (dense, dynamique)
- Le contenu encyclopédique dicte la hiérarchie visuelle
- Chaque interaction révèle une couche de profondeur

**Color Philosophy** :
- Fond crème chaud `#F5F0E8` — évoque le papier vieilli des encyclopédies
- Encre noire profonde `#1A1209` — typographie principale
- Or académique `#C9A227` — accent primaire, récompenses, XP
- Rouge vermillon `#C0392B` — alertes, erreurs, urgence (timer)
- Vert sauge `#4A7C59` — succès, validation

**Layout Paradigm** : Grille éditoriale asymétrique — colonnes de largeurs inégales, textes en débordement, éléments ancrés à gauche avec espace blanc délibéré à droite. Navigation horizontale fixe ultra-fine.

**Signature Elements** :
- Numérotation en chiffres romains pour les chapitres
- Lignes de séparation fines en or sur fond crème
- Cercles du mindmap avec bordure double (trait fin intérieur + trait épais extérieur)

**Interaction Philosophy** : Chaque action a un poids — les clics produisent des transitions lentes et délibérées comme tourner une page. Le système XP est révélé progressivement, jamais crié.

**Animation** : Transitions de page par glissement horizontal (comme feuilleter un livre). Les cercles du mindmap apparaissent en cascade avec un léger rebond. Le timer se vide avec une animation d'encre qui s'efface.

**Typography System** :
- Display : Playfair Display (serif élégant pour titres)
- Body : Source Serif 4 (lisibilité encyclopédique)
- UI/Labels : Space Mono (monospace pour les données XP, niveaux, codes)

## Approche 2 : Terminal du Savoir (Dark Academic Gamifié)

**Design Movement** : Dark Academic Digital — l'esthétique des salles de lecture nocturnes rencontre l'interface d'un jeu de rôle textuel.

**Core Principles** :
- Fond sombre profond avec lumière focale sur le contenu actif
- La gamification est visible mais jamais vulgaire — XP et niveaux intégrés dans l'architecture
- Contraste lumineux fort : texte clair sur fond très sombre
- Sentiment d'exploration et de découverte à chaque écran

**Color Philosophy** :
- Fond `#0F0E0A` — noir chaud, presque sépia inversé
- Surface cartes `#1C1A14` — brun très sombre
- Or antique `#D4A843` — accent principal, XP, récompenses
- Ivoire `#EDE8D5` — texte principal
- Bordeaux `#8B2635` — erreurs, temps écoulé
- Vert émeraude `#2D6A4F` — succès, déblocage

**Layout Paradigm** : Interface en couches — le contenu principal occupe 70% de l'écran, avec des panneaux latéraux semi-transparents pour le contexte (XP, progression). Les cercles du mindmap flottent dans un espace sombre avec des connexions lumineuses.

**Signature Elements** :
- Effets de lueur (glow) dorée sur les éléments actifs et débloqués
- Textures de parchemin subtiles en overlay sur les fonds
- Particules légères qui s'animent lors des gains d'XP

**Interaction Philosophy** : L'interface réagit comme un organisme vivant — les cercles verrouillés pulsent légèrement, les éléments débloqués s'illuminent avec un flash doré.

**Animation** : Gain d'XP avec particules dorées montantes. Déblocage de cercle avec animation de "révélation" (le cadenas se brise, la lumière se répand). Timer avec cercle SVG qui se vide en rouge.

**Typography System** :
- Display : Cormorant Garamond (serif dramatique pour titres)
- Body : Crimson Pro (serif lisible pour textes)
- UI : JetBrains Mono (monospace pour données techniques)

## Approche 3 : Cartographie du Savoir (Exploration Géographique)

**Design Movement** : Neo-Cartographic Editorial — l'interface comme une carte ancienne interactive, où chaque sujet est un territoire à explorer.

**Core Principles** :
- L'apprentissage comme voyage géographique — on "explore" des territoires de connaissance
- Fond texturé évoquant le papier de carte topographique
- Hiérarchie visuelle inspirée des légendes cartographiques
- Les connexions entre concepts sont des routes tracées

**Color Philosophy** :
- Fond `#F2EBD9` — papier de carte ancien
- Encre cartographique `#2C3E50` — texte principal
- Or de boussole `#B8860B` — accent principal
- Bleu océan `#1A5276` — liens, navigation
- Vert territoire `#1E8449` — succès, zones explorées
- Rouge danger `#922B21` — erreurs, zones verrouillées

**Layout Paradigm** : Les cercles du mindmap sont disposés comme des villes sur une carte, reliés par des lignes pointillées. La progression révèle de nouvelles "régions" du savoir.

**Signature Elements** :
- Lignes de topographie en arrière-plan subtil
- Icônes de boussole et de sextant pour la navigation
- Les cercles verrouillés sont des "terra incognita" avec texture de brouillard

**Interaction Philosophy** : Chaque cercle débloqué "révèle" une zone de la carte — animation de brouillard qui se dissipe. Le quiz est une "épreuve d'exploration".

**Animation** : Révélation de carte au scroll. Cercles qui se dévoilent comme des îles émergentes. XP qui s'accumule comme une jauge de "territoire exploré".

**Typography System** :
- Display : Libre Baskerville (serif classique pour titres)
- Body : Lora (serif chaleureux pour textes)
- UI : IBM Plex Mono (monospace propre pour données)
